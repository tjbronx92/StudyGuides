\section{Latex Base Classes}
\begin{itemize}
    \item article
    \item report
    \item book
    \item letter
    \item slides
    \item beamer
        \begin{itemize}
            \item beamer is a LATEX class for creating presentations that are held using a projector, but it can also be
used to create transparency slides.
    \end{itemize}
    \item moderncv
        \begin{itemize}
            \item document class designed for creating modern curricula vitae (CVs) and cover letters, offering both simplicity and customization.
 It provides five built-in styles: classic, casual, banking, oldstyle, and fancy, with the ability to define custom styles by modifying colors, fonts, and icons.
    \end{itemize}
    \item seminar
        \begin{itemize}
            \item a LaTeX document style for typesetting slides or
  transparencies, and accompanying notes.
    \end{itemize}
\end{itemize}
