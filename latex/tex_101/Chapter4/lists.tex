\section*{Useful packages}
Latex provides several packages for designing the layout:
    \begin{itemize}
        \item Page layout
            \begin{itemize}
                \item \texttt{geometry}: Adjust page margins and dimensions.
                \item \texttt{typearea}: Optimize page layout for different paper sizes.
            \end{itemize}
        \item Headers and footers
            \begin{itemize}
                \item \texttt{fancyhdr}: Customize headers and footers.
                \item \texttt{scrlayer-scrpage}: Advanced header/footer management (KOMA-Script).
            \end{itemize}
        \item Line spacing
            \begin{itemize}
                \item \texttt{setspace}: Adjust line spacing.
            \end{itemize}   
    \end{itemize}
