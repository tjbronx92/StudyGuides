\section*{Useful packages}
Latex provides several packages for designing the layout:
\begin{itemize}
    \item \texttt{fancyhdr}: Customize headers and footers.
    \item \texttt{geometry}: Adjust page margins and dimensions.
    \item \texttt{titlesec}: Modify section and chapter headings.
    \item \texttt{multicol}: Create multi-column layouts.
    \item \texttt{setspace}: Control line spacing (e.g., single, one-and-a-half, double).
\end{itemize}
